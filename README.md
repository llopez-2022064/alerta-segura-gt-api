# Alerta Segura GT

Una aplicación web para que las personas en Guatemala puedan reportar de forma **anónima** extorsiones, asaltos, amenazas, robos y otros delitos.

El objetivo es ayudar a la comunidad a estar más informada sobre las zonas de mayor riesgo y facilitar que más personas denuncien de manera segura y rápida.

### ¿Qué puedes hacer en Alerta Segura GT?

- Reportar un incidente de forma anónima (no se necesita registrarse)
- Ver en un mapa de Guatemala las zonas más afectadas, con colores que indican el nivel de peligro
- Adjuntar fotos o videos como evidencia (opcional)
- Consultar estadísticas simples de seguridad por departamento

Pensada especialmente para ayudar a las familias y personas que viven en Guatemala a tomar mejores decisiones y estar más alertas.
